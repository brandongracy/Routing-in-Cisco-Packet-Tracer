# Routing-in-Cisco-Packet-Tracer
## Routing is a foundational concept in networking that involves directing data packets from one network to another based on their destination IP addresses. In cybersecurity, understanding routing is essential because routes dictate how data flows between devices and networks, which impacts both network performance and security. Misconfigured routes or vulnerabilities in routing protocols can expose networks to threats, such as route hijacking, data interception, or Denial of Service (DoS) attacks. By mastering routing, cybersecurity professionals can protect data as it moves across networks and minimize potential attack vectors. This lab builds upon the previous DHCP lab, adding a router to allow communication between different VLANs using DHCP.
